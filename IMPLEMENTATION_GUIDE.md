## 零成本启动步骤

### 1. GitHub仓库设置（免费）
```bash
# 在GitHub创建新仓库
仓库名: ai-income
描述: AI自动化收入项目
公开仓库
```

### 2. 推送代码到GitHub
```bash
cd ~/.openclaw/workspace/ai-income-project
git remote add origin https://github.com/你的用户名/ai-income.git
git branch -M main
git push -u origin main
```

### 3. 启用GitHub Pages（免费）
1. 进入仓库设置 → Pages
2. 分支选择: `gh-pages`（GitHub Actions会自动创建）
3. 保存设置
4. 访问地址: `https://你的用户名.github.io/ai-income/`

### 4. 收入渠道注册（全部免费）

#### Google AdSense
1. 访问: https://www.google.com/adsense
2. 使用Google账号登录
3. 添加网站: `https://你的用户名.github.io/ai-income/`
4. 获取广告代码
5. 添加到网站模板

#### Amazon联盟
1. 访问: https://affiliate-program.amazon.com
2. 注册账号
3. 获取推广链接
4. 在文章中插入相关产品链接

#### Buy Me a Coffee（捐赠）
1. 访问: https://buymeacoffee.com
2. 注册账号
3. 获取捐赠按钮代码
4. 添加到网站页脚

### 5. 流量分析（免费）
#### Google Analytics
1. 访问: https://analytics.google.com
2. 创建媒体资源
3. 获取跟踪ID: `G-XXXXXXXXXX`
4. 添加到`config.toml`:
```toml
googleAnalytics = "G-XXXXXXXXXX"
```

### 6. 自动化设置

#### 本地cron job（每天自动运行）
```bash
# 编辑crontab
crontab -e

# 添加以下行（每天凌晨2点运行）
0 2 * * * cd /root/.openclaw/workspace/ai-income-project && ./scripts/deploy.sh >> /tmp/ai-income.log 2>&1
```

#### GitHub Actions（已配置）
- 自动每天运行
- 自动生成内容
- 自动部署更新
- 无需人工干预
