# 一行命令部署AI自动化收入项目

## 🚀 最简单的部署方式

只需运行这一行命令，然后按照提示操作：

```bash
cd ~/.openclaw/workspace/ai-income-project && ./scripts/auto_deploy.sh
```

## 📋 命令将帮你完成：

1. **检查Git配置**（如未配置会提示输入）
2. **显示创建GitHub仓库的具体步骤**
3. **引导你输入仓库地址**
4. **自动推送代码**
5. **显示后续操作说明**

## ⏱️ 预计时间：5分钟

### 你的操作流程：
1. **运行上面的命令**
2. **根据提示在浏览器访问** https://github.com/new
3. **创建仓库**（名称：`ai-income`）
4. **复制仓库地址**粘贴到终端
5. **按照脚本提示完成认证**（如果需要）

### 脚本完成后需要你手动操作：
1. 在GitHub仓库设置中启用Pages（点击几下）
2. 等待2-3分钟让网站生效

## 🌐 完成后你将获得：

1. **网站地址**：`https://你的用户名.github.io/ai-income/`
2. **自动化系统**：每天自动更新内容
3. **收入追踪**：自动生成收入报告
4. **零成本运营**：全部免费服务

## 💡 为什么选这个方案？

- ✅ **最简单**：只需运行一个命令
- ✅ **最安全**：不需要提供任何敏感信息
- ✅ **最快**：5分钟完成部署
- ✅ **最可靠**：脚本处理所有技术细节

## 🏁 立即开始

复制粘贴这条命令到终端：

```bash
cd ~/.openclaw/workspace/ai-income-project && ./scripts/auto_deploy.sh
```

然后按照屏幕提示操作！

## 📞 遇到问题？

1. **认证失败**：使用GitHub token作为密码
2. **网络问题**：检查网络连接
3. **权限问题**：确保有创建仓库的权限
